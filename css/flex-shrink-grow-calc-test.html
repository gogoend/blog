元素最终宽度 = ( 父元素width - 所有子元素width之和 ) * ( 当前子元素flex-grow / 所有元素flex-grow之和 ) + 元素width

以第一个元素为例：
(600 - 130) * (2 / 16) + 40 = 98.75
以第三个元素为例：
(600 - 130) * (4 / 16) + 40 = 157.5
以第四个元素为例：
(600 - 130) * (8 / 16) + 10 = 245
